## Основные директории

*   **`src/`** — корневой каталог исходного кода.
   *   `src/components/` — React-компоненты экранов (лобби, раунды, результаты).
   *   `src/game/` — логика игры (состояние, правила расчёта позиций и призов).
   *   `src/network/` — сетевое взаимодействие (WebSocket-клиент).
   *   `src/questions/` — база вопросов и детерминированный выбор по seed.

## Основные модули

1.  **Модуль инициализации** (`main.tsx`) — точка входа, загрузка корневого компонента.
2.  **Модуль роутинга** (`App`, `Router`) — переключение экранов (Главная, Лобби, Раунды, Результаты).
3.  **GameState** (`src/game/state.ts`) — структура состояния (игроки, раунд, позиции) и функции его обновления.
4.  **GameLogic** (`src/game/logic.ts`) — правила раундов (Cash Builder, Head-to-Head, Final Chase).
5.  **WebSocketClient** (`src/network/websocket.ts`) — подключение, отправка/приём событий, обработка разрывов.
6.  **QuestionBank** (`src/questions/bank.ts`) — локальная база вопросов и генерация последовательности по seed.
7.  **Компоненты экранов:**
   *   `MainScreen` — главная страница (создать/присоединиться к игре).
   *   `LobbyScreen` — лобби с кодом приглашения и списком игроков.
   *   `CashBuilderScreen`, `HeadToHeadScreen`, `FinalChaseScreen` — интерфейсы раундов.
   *   `ResultsScreen` — итоговый результат, статусы игроков.
